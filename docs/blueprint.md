# **App Name**: SkillForge

## Core Features:

- Skill Definition Editor: A user interface to input skill metadata: name, purpose, and detailed description, framing the problem the skill solves.
- Input & Output Schema Builder: Interactive forms to define the data structures (inputs and expected outputs) for the skill, aligning with what an AI agent would send/receive.
- AI-Generated SKILL.md Examples: An AI tool that automatically generates usage examples and instructional content based on the defined skill, inputs, and outputs.
- SKILL.md Document Export: Functionality to compile all skill details, schemas, and AI-generated examples into a correctly formatted SKILL.md file for download.
- Skill Draft Management: A dashboard to list, save, load, and manage multiple skill definitions locally within the browser during the development process.

## Style Guidelines:

- The application employs a dark color scheme, suggesting a focused and sophisticated environment suitable for development tasks.
- Primary interactive elements will use a deep, contemplative purple-blue (#815EFF), reflecting innovation and precision.
- The background will feature a muted, very dark purple-gray (#17151D), providing a calm canvas that allows content to stand out.
- An accent color of vibrant, clear blue (#76BBF7) will highlight key information, calls to action, and indicate interactivity.
- Headlines and body text will utilize 'Inter' (sans-serif) for its modern, objective, and highly legible appearance, suitable for technical documentation.
- Code snippets, schema definitions, and any structured technical content will be presented in 'Source Code Pro' (monospace) to ensure clarity and easy readability of syntax.
- Clean, line-based icons will be used to represent actions and data elements, maintaining a professional and modern aesthetic consistent with a developer tool.
- A clean, responsive, multi-column layout is designed to efficiently present complex forms and generated documentation, optimizing for both wide displays and smaller screens.
- Subtle transitions and hover effects will provide feedback on user interactions without distracting from the core content, enhancing the overall user experience.
